Gear On — Automotive E-Commerce UI/UX Case Study

Overview:
Gear On is a modern automotive e-commerce UI/UX case study designed to create a seamless digital shopping experience for users searching for automotive accessories, vehicle products, and performance equipment. The project focuses on improving usability, visual consistency, and product discoverability through a clean and structured interface system.

Challenge:
Existing automotive shopping platforms often contain cluttered layouts, confusing navigation patterns, and inconsistent user flows that make product exploration difficult. The objective of this project was to redesign the experience into a modern, user-friendly interface with improved accessibility and simplified interaction patterns.

Design Approach:
The process included competitor research, wireframing, user flow planning, high-fidelity UI design, and interactive prototyping. Special attention was given to spacing systems, typography hierarchy, component consistency, and responsive layout structure to create a premium user experience.

Key Highlights:
• Modern automotive UI system
• Improved product browsing experience
• Clean visual hierarchy
• User-focused navigation flow
• Consistent spacing and component system
• Responsive and accessibility-focused design

Tools Used:
Figma, UX Research, Wireframing, User Flow, Prototyping, Visual Design System

03

VHS Hospital — Website Redesign

A complete UX redesign of the VHS Hospital website to make healthcare information more accessible, trustworthy, and easy to navigate for patients and visitors.

Overview: VHS Hospital needed a redesign of their existing website to better serve patients searching for medical services, doctor information, appointment booking, and department details online.

Problem: The original website had a cluttered layout, poor information hierarchy, and confusing navigation that made it difficult for patients to quickly find critical details — such as specialities, doctor profiles, and contact information. In a healthcare context, this creates frustration and erodes patient trust.

Solution: Restructured the entire information architecture around patient needs. Introduced clear service categories, a simplified navigation system, and a prominent appointment and contact flow to reduce friction for first-time visitors.

Design Approach: Applied a clean, professional UI with a calm and accessible colour palette suited for healthcare. Focused on readability with proper font sizing, generous whitespace, and a structured layout. Wireframes were built first to validate the structure before high-fidelity design in Figma.

Outcome: Delivered a more trustworthy, accessible, and patient-friendly hospital website that makes it significantly easier for users to find services, locate doctors, and connect with the hospital.

Tools Used: Figma, UX Audit, Wireframing, Information Architecture, Visual Design

SI Homes — Real Estate Website Redesign

A professional and user-friendly website redesign for SI Homes, a modern real estate company specialising in residential properties.

Overview: SI Homes required a complete website redesign to reflect their modern brand identity and help potential buyers and renters easily explore residential properties, understand their services, and connect with the team.

Problem: The existing website lacked the professional aesthetic expected in the real estate sector. The layout was outdated, property listings were hard to browse, and the overall experience failed to communicate trust and credibility to prospective clients.

Solution: Redesigned the website with a premium, modern UI tailored for real estate. Introduced structured property listing layouts, clear service sections, an intuitive search and filter experience, and a strong call-to-action flow to drive enquiries.

Design Approach: Focused on a clean, spacious layout with high visual impact — using bold typography, professional imagery placement, and a refined colour system that reflects the brand's positioning. Every section was designed to build trust and guide the user toward taking action.

Outcome: Delivered a polished, professional real estate website that strengthens SI Homes' brand presence, improves property discoverability, and creates a seamless experience for buyers and renters browsing residential listings.

Tools Used: Figma, UI Design, Component System, User Flow, Visual Branding

05

Premium Cinematic Makeup Artist — Portfolio Website

A premium cinematic portfolio website redesign for a professional makeup artist, crafted to showcase her artistry and attract high-end clients.

Overview: Designed and redesigned a complete portfolio website for a professional makeup artist, with a focus on cinematic aesthetics, strong visual storytelling, and a premium brand experience that reflects the quality of her work.

Problem: The existing website did not reflect the professional level and artistic quality of the makeup artist's work. It lacked visual impact, had no clear service structure, and failed to create the premium impression needed to attract high-end clientele in the beauty and film industry.

Solution: Redesigned the website with a cinematic, luxury aesthetic — using dramatic visuals, elegant typography, and a dark, immersive layout that lets the portfolio images take centre stage. Structured the site around services, portfolio gallery, testimonials, and a clear booking flow.

Design Approach: Adopted a visual-first, editorial design approach inspired by high-end beauty and film industry aesthetics. Every element — from font choices to section spacing — was crafted to communicate luxury, artistry, and professionalism. Navigation was kept minimal to keep focus on the work.

Outcome: Delivered a visually stunning, premium portfolio website that powerfully represents the artist's brand, elevates her perceived value, and creates a compelling experience that converts visitors into enquiries and bookings.

Tools Used: Figma, Visual Branding, Editorial Design, UX Flow, Typography System
